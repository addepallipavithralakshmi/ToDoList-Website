Do List Web Application (MERN Stack)

A simple and efficient To-Do List web application built using the React framework for the frontend, MongoDB for storing tasks, and ExpressJS + Node.js as the backend server. This project demonstrates full-stack development using the MERN stack, including task creation, deletion, and management through a clean UI and responsive API.
